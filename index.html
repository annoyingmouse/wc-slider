Adapted from the original JSFiddle.

Installation

<script type="module" src="https://unpkg.com/@annoyingmouse/wc-slider/wc-slider.js"></script>

Alternatively:

<script type="module" src="https://cdn.skypack.dev/@annoyingmouse/wc-slider/wc-slider.js"></script>

You can use it minified by adding using this instead:

<script type="module" src="[WHATEVER CDN]/@annoyingmouse/wc-slider/dist/wc-slider.min.js"></script>

Please remember to change [WHATEVER CDN] to your favourite CDN

Examples

Basic Example (my original colours)

Changing width attribute randomly (100%) every ten seconds between 50% and 100%

Changing min and max attributes randomly (1, 7) every ten seconds

Changing from and to attributes randomly (#afd6f3, #2b91de) every ten seconds

Changing deselected-from and deselected-to attributes randomly (#d0d5d5, #7f8c8d) every ten seconds
